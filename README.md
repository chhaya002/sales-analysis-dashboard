# 📊 Sales Analysis Dashboard

An interactive Excel dashboard built to analyze sales performance and identify business insights from nearly 10,000 sales records.

## Objectives
- Analyze sales trends
- Identify top customers
- Evaluate profitability
- Study regional sales performance

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Dashboard Features
- Sales by Category
- Monthly Sales Trend
- Profit Analysis
- Top Customers
- Sales by State
- Customer Insights

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Data Visualization
- Dashboard Design
- Reporting
